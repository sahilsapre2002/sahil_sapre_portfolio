## 🚀 Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- Git (optional, for version control)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/sahilsapre2002/sahil_sapre_portfolio.git
   cd sahil_sapre_portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

3. **Access the site**
   - Navigate to `http://localhost:8000` (or the port shown)

### No Build Process Required
This is a static website with no dependencies or build steps. It uses vanilla HTML, CSS, and JavaScript.

## 🌐 Deployment

### Netlify (Current Deployment)

The site is configured for automatic deployment on Netlify:

1. **Automatic Deployment**: 
   - Connected to GitHub repository
   - Auto-deploys on push to main branch

2. **Configuration**: See `netlify.toml` for settings
   - Build command: None (static site)
   - Publish directory: Root
   - Custom redirects configured

3. **Manual Deployment**:
   ```bash
   # Install Netlify CLI
   npm install -g netlify-cli
   
   # Deploy
   netlify deploy --prod
   ```

### Alternative Platforms

**GitHub Pages:**
```bash
# Enable GitHub Pages in repository settings
# Select main branch and root folder
```

**Vercel:**
```bash
npm install -g vercel
vercel --prod
```

**Custom Server:**
- Upload all files to web server root
- Ensure `index.html` is the entry point
- No special server configuration needed

## 🛠️ Customization

### Changing Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --primary: #00F5D4;    /* Your primary color */
    --accent: #7CFF01;     /* Your accent color */
    --bg-deep: #020617;    /* Background color */
}
```

### Updating Content
1. **Personal Info**: Edit text in `index.html` sections
2. **Experience/Education**: Modify timeline items
3. **Skills**: Update skill badges in skills section
4. **Projects**: Add/remove project cards
5. **Social Links**: Update href attributes in hero and contact sections

### Adding Sections
1. Add new `<section>` in `index.html`
2. Add navigation link in navbar
3. Style the section in `styles.css`
4. Add scroll reveal animation if desired

### Profile Image
- Replace image source in hero section
- Update fallback URL in `onerror` attribute
- Recommended size: 300x300px
